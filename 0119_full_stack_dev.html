___# 0119 notes: full stack development
---------------------
Stacks -->
----------combo of software apps and components used to fix a problem or create a project

front-end stack -> main focus is UI (user interface)
| .. languages used: HTML/CSS, JavaScript

back-end stack -> main applications core
| .. languages: Python, JavaScript, django rest framework (DRF),

data stack -> contains SQL db engines, caching tools (Redis app), and tools required to process data
| .. MySQL, MariaDB, PostgreSQL


__# N-tier architecture
---------------------the parts must be physically seperated & stay on different infrastructure servers
Layers -> virtual seperations of different parts
Tiers -> physical seperations of different parts

| tier1 ex.data server,
| tier2 app server,
| tier3 client pc ..aka. presentation tier ..aka. thin clients

3 tiers:
Presentation, Application, Data
client tier > data tier > presentation tier >
