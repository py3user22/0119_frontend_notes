__#0120 block and inline elements
---------------------------
block level elements
| .. stack on top and on new lines

inline elements
| .. will stay on same line and go horizontal

__#notes
-------------------
< span> < /span> is for inline text input, to keep on the same line horizontally

< div> < /div> is for block level, will seperate to own line the text in it

< body>
| .. < div>
| ...... < span> < /span>
| ...... < span> < /span>
| ...... < span> < /span>
| .. < /div>
< /body>
